如何取得金鑰 (Token)?

  1. 登入 GitHub，點擊右上角頭像 > Settings。
  2. 左側選單最下方 > Developer settings。
  3. 選 Personal access tokens > Fine-grained tokens。
  4. 點擊 Generate new token：
    • Repository access: 選 Only select repositories (選擇本專案)。
    • Permissions: 點開 Contents，選 Read and write。
  5. 複製產生的 github_pat_... 字串即為 Token。
